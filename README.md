# TanStack Start LWA Demo

## Description
TanStack Start を Lambda Web Adapter で動作させる最小構成のデモプロジェクトです。

## Status
PoC / Experimental

## QuickStart

1. DevContainer で起動
2. 依存関係のインストール
   ```bash
   pnpm install
   ```
3. ビルド
   ```bash
   pnpm build
   ```
4. デプロイ
   ```bash
   pnpm cdk:deploy
   ```
5. 出力された Function URL にアクセスして動作確認
